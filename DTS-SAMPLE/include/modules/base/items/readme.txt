命名规则：
重要道具类型可以以英文名命名，乱七八糟的道具请以itmk命名，同类物品请一起处理。
由于命名空间不区分大小写，请在以itmk命名时，第一个字母如果是大写，加前缀“u”，否则加前缀“l”。

例：
	补给（类型H）作为重要游戏机制，应命名为“edible”而不是“item_uh”；各类补给应该一起处理，而不是HB,HS,HH各写一个模块
	陷阱（类型T）作为重要游戏机制，应命名为“trap”而不是“item_ut”
	强化药物（类型M）仅仅是一种道具，所以应命名为“item_um”

代码风格：
	强烈建议接管itemuse时写成类似
	if ($itmk[0] == 'M')
	{
		useitem_um($theitem);
		return;
	}
	$chprocess($theitem);
	这样的形式。
	
	开一个新函数使用你的物品方便未来的人去接管。
